## Radio button
Radio buttons are used to distinguish between mutually exlusive choices
in the context of a form.

<Playground>
  <Fragment>
    <RadioButton name="fruit" value="true" id="pineapples" checked onToggle={() => {}}>
      <Label htmlFor="pineapples">Pineapples</Label>
    </RadioButton>
    <RadioButton name="fruit" value="bananas" id="bananas" onToggle={() => {}}>
      <Label htmlFor="bananas">Bananas</Label>
    </RadioButton>
  </Fragment>
</Playground>

### Best practices
- Radio button choices should represent mutually exclusive options.

### Usage guidelines
- Do not use radio butons for "yes/no" or "on/off" decisions. Instead use
  the [Toggle](/components/toggle).
